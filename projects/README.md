# 📊 Customer Analysis SQL Project

## 📌 Project Overview
This project demonstrates SQL skills by analyzing customer and order data.

The goal is to:
- Understand customer behavior
- Calculate revenue
- Perform data cleaning
- Practice real-world SQL queries

---

## 🗂️ Dataset Description

### Customers Table
- customer_id – Unique ID
- name – Customer name
- city – Customer location

### Orders Table
- order_id – Unique order ID
- customer_id – Links to customers
- product – Product name
- amount – Order value

---

## 🔍 Key SQL Concepts Used

- JOINs
- GROUP BY
- HAVING
- Aggregate Functions (SUM, COUNT)
- Data Cleaning (TRIM, UPPER)
- Filtering (WHERE)

---

## 📈 Analysis Performed

### 1. Total Spending per Customer
Finds how much each customer has spent.

### 2. High Value Customers
Identifies customers spending more than 1000.

### 3. Revenue per Product
Shows which products generate the most revenue.

### 4. Data Cleaning
Standardizes customer names using TRIM and UPPER.
